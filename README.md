## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aimadhav/startup.git
cd startup
```

2. Install dependencies:
```bash
npm install
```

3. Build the project:
```bash
npm run build
```

4. Start the server:
```bash
npm start
```

Or run in development mode:
```bash
npm run dev
```

5. Open your browser and navigate to:
```
http://localhost:3000
```
